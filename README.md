1. Data Preparation and Preprocessing

1.1 RAF Dataset

- Create a new directory called "RAF_raw" in the project's data directory.
- Place the "EmoLabel" and "Image" directories from the official database in "RAF_raw".
- IMPORTANT: To use our self developed alignment method, delete the "aligned" directory in "Image".
- Run the preprocessing script:
    python src/preprocess_RAF_raw.py
- Run the preparation script:
    python src/prepare_RAF_raw.py
=> Data directory after these 1.1:
                                      ├── data/
                                          ├── RAF_raw/                # Place original dataset here (EmoLabel & Image folders)
                                          ├── RAF_aligned_processed/  # Generated by prepare_RAF_raw.py (used for training)
                                          └── RAF_original_processed/ # Generated by prepare_RAF_raw.py (full size images)

1.2 KDEF Dataset

- Place the official dataset "KDEF" in the "data" directory.
- Run the preparation script:
    python src/prepare_KDEF.py
- Run the preprocessing script:
    python src/preprocess.py
=> Data directory after 1.1 and 1.2:
                                      ├── data/
                                          ├── KDEF/                   # Place original dataset here
                                          │     ├── AF01/   ##           
                                          │    ...          # Artifacts of the original KDEF dataset. Contain neutral faces, fullsided faces and missing pictures
                                          │     ├── BM35/   ##
                                          │     ├── EmoLabel/
                                          │     └── Image/
                                          │           ├── KDEF_aligned_processed/   # Generated by prepare_RAF_raw.py (used for training)
                                          │           └── KDEF_original_processed/  # Generated by prepare_RAF_raw.py (full size images)
                                          ├── RAF_raw/                
                                          ├── RAF_aligned_processed/ 
                                          └── RAF_original_processed/

1.3 Merging

- In order to use KDEF as additional train/test (currently only train) data, add the aligned+processed pictures of KDEF to the aligned+proccessed pictures of RAF
- Run the merging scrip:
